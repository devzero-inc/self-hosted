# DevZero Self-Hosted - Terraform - AWS - Cluster Extensions

This Terraform configuration is designed to deploy Cluster Extensions for an Amazon EKS cluster. These extensions include optional components like the Cluster Autoscaler and other utilities that enhance the functionality and performance of your Kubernetes cluster.
